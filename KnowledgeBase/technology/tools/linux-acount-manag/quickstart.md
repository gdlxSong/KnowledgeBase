### 创建分组

>https://man.linuxde.net/groupadd

```bash
$ groupadd -g 5555 xgdl
# -g 用于指定组id
```

### 创建用户

>https://man.linuxde.net/useradd

```bash
$ useradd -g xgdl -m gdl
# -g 指定用户组
# -m自动创建用户登录目录
```
